# PixelKnight
თამაშის სიუჟეტი დაფუძნებულია რაინდის და ჯადოქრის დაპირისპირების ისტორიაზე. რაინდი რომელმაც დაკარგა სამეფო, შური უნდა იძიოს ჯადოქარზე და დაიბრუნოს ყველაფერი რაც მას ეკუთვნის. რაინდს მოუწევს რთული ეტაპის გავლა, ბევრი დაბრკოლების გადალახვა და სამართლიანობის აღდგენისათვის მტრის განადგურება, მიზნის მისაღწევად. თამაშის საბოლოო მიზანი სამეფოს (ქვეყნის) დაცვაა ჯადოქრისგან, რომელთან მისასვლელი გზა არ იქნება მარტივი.

## რას გვასწავლის PixelKnight
PixelKnight გვასწავლის სიბოროტის დამარცხებას,სამართლიანობას,იმას თუ როგორ მოვიქცეთ კონკრეტულ სიტუაციაში, როგორ დავამარცხოთ სიბოროტე და როგორ დავიბრუნოთ ის რაც ჩვენ გვეკუთვნოდა.
